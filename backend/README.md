# Backend

## Ejecución

Ejecutar desde src/ :
`uvicorn main:app --reload`

## Rutas

/data: Obtiene info de temperatura/humedad
/data/historiacl: Obtiene el historial de datos de la base de datos
/control: Controla los parámetros para regar, etc
